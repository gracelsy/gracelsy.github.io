EasyTier异地组网教程
发表于2025-04-23|更新于2025-05-13|LinuxDebianUbuntu|EasyTier•Openwrt•Docker•异地组网
异地组网简单来说，就是把两个或多个不在同一地方的小网络（比如家里和公司），通过互联网或专用线路连接成一个大网络。这样不同地方的电脑、打印机、服务器等设备就能互相访问，就像在同一个办公室一样方便。当下最常用的异地组网工具有zerotier、Tailscale、wireguard，当然还有其他组网工具。 EasyTier 是大佬 @雨过。天晴 开源的内网穿透远程组网方案。EasyTier最主要的特点就是去中心化，客户端也是DERP 中继服务器（当你通过命令或配置文件开放相应的端口启动后，EasyTier就是服务端）。 Debian/Ubuntu安装EasyTier访问 ⬇️下载页面 下载适用于您操作系统的二进制文件或安装包。 注：请下载最新的二进制文件或安装包不要复制粘贴 12345678910# 更新软件源并安装wget、unzipapt-get update&&apt install wget unzip -y# 在/etc文件夹中创建easytier，并切换至/etc/easytiermkdir -p /etc/easytier && cd...
